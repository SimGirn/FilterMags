# FilterMags:Code/Astro Project
The `FilterMags` is a python package to calculate the AB magnitude of an astronomical object in various filters for spectra from SDSS. For the detailed math refer to this link https://mfouesneau.github.io/pyphot/photometry.html. 

## How to install and run FilterMags?
The FilterMags can be installed using the pip or by following the link https://pypi.org/project/FilterMags/0.0.6/

```bash
!pip install FilterMags==0.0.6
```

To run it we need to import as follows:

```bash
import FilterMags as fm
```
